# 📊 بوت فرص السوق الأمريكي — استراتيجية شمعة الافتتاح

سكانر يفحص أسهم بورصة نيويورك ويبحث عن الأسهم التي تنطبق عليها
استراتيجية **Opening Range Breakout + VWAP**، ويعرض النتائج في لوحة تحكم ويب.

---

## 🧠 الاستراتيجية (نفس منطق مؤشر Pine Script)

| القاعدة | الشرط |
|---|---|
| شمعة الافتتاح | أول شمعة 5 دقائق بعد 9:30 صباحاً بتوقيت نيويورك |
| **شراء** | إغلاق الشمعة فوق **قمة شمعة الافتتاح** *و* فوق **VWAP** |
| **بيع** | (وأنت في صفقة) إغلاق تحت **VWAP** والشمعة السابقة كانت فوق القمة |
| **إعادة شراء** | إغلاق فوق **قمة شمعة البيع** *و* فوق VWAP |
| الأهداف | القمة + (2× / 4× / 6× / 8× / 10×) مدى شمعة الافتتاح |
| فلتر | موقع السعر من EMA 200 (يُعرض للمساعدة فقط) |

> الإشارة تُحسب على **إغلاق** الشمعة فقط — مطابق لـ `barstate.isconfirmed` في Pine Script.

## 🚀 التشغيل على جهازك

```bash
pip install -r requirements.txt

python scan.py                       # فحص كل الأسهم في tickers.txt
python scan.py AAPL NVDA TSLA        # فحص أسهم محددة
python scan.py --date 2026-07-21     # فحص جلسة يوم سابق (لتجربة الاستراتيجية)
```

> **ملاحظة:** في أول 10–15 دقيقة من فتح السوق سترى «بانتظار شموع كافية» —
> هذا طبيعي، لأن الاستراتيجية تحتاج شمعة الافتتاح + شمعة إغلاق بعدها على الأقل.

ثم لعرض اللوحة (لا تفتح `index.html` مباشرة — لازم سيرفر):

```bash
python -m http.server 8777 --directory docs
```

وافتح المتصفح على <http://localhost:8777>

## ☁️ التشغيل التلقائي المجاني 24/7

1. أنشئ مستودعاً **عاماً (Public)** جديداً على GitHub.
   > مهم: المستودعات العامة لها دقائق تشغيل **غير محدودة** مجاناً.
   > الخاصة محدودة بـ 2000 دقيقة/شهر ولن تكفي للفحص كل 5 دقائق.

2. ارفع هذا المجلد إليه:
   ```bash
   git init
   git add .
   git commit -m "أول إصدار"
   git branch -M main
   git remote add origin https://github.com/USERNAME/REPO.git
   git push -u origin main
   ```

3. فعّل **GitHub Pages**: `Settings → Pages → Source: main branch → /docs`
   وسيصبح رابط لوحتك: `https://USERNAME.github.io/REPO/`

4. الفحص التلقائي يبدأ فوراً حسب `.github/workflows/scan.yml`
   (كل 5 دقائق خلال أيام وساعات التداول).
   لتجربته يدوياً: تبويب **Actions** → اختر الـ workflow → **Run workflow**.
